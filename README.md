**Stickify** — мини-приложение ВКонтакте для анализа коллекций стикеров, просмотра статистики и рейтингов.

Интерфейс выполнен в тёмно-синем неоновом стиле с glassmorphism, плавными анимациями и персонажами бренда Stickify.

## Возможности

| Функция | Описание |
|--------|----------|
| **Анализ коллекции** | Вставка ссылки на профиль VK или выбор друга через VK Bridge |
| **Профиль коллекционера** | Имя, ссылка, уровень и аватар из VK (`VKWebAppGetUserInfo`) |
| **Сетка стикеров** | Просмотр наборов с обложками и редкостью |
| **Редкие наборы** | Отдельный блок Premium / rare / epic / legendary |
| **Статистика** | Сводка по наборам, голосам и рублям + графики динамики |
| **TOP-100** | Рейтинг коллекционеров с поиском и сортировкой |
| **Share-карточка** | Вертикальная карточка 1080×1920 в стиле VK |
| **Экспорт PNG** | Снимок видимой карточки через `html2canvas` |
| **VK Stories** | Публикация карточки в истории через `VKWebAppShowStoryBox` |
| **Онбординг** | Пошаговое знакомство с приложением (показывается один раз) |

## Разделы приложения

<img width="1820" height="1000" alt="2026-05-29_10-59-14" src="https://github.com/user-attachments/assets/cbd19cdc-4c58-4e36-b565-fbb629231a44" />

Навигация — нижняя панель из 5 вкладок.

### Главная

- Поле ввода ссылки на профиль VK (`https://vk.com/id…`).
- Кнопка **«Вставить ссылку»** — запуск анализа коллекции.
- Кнопка **«Добавить друга»** — выбор друга через `VKWebAppGetFriends`.
- Блок **«После анализа»** с краткой статистикой:
  - количество наборов;
  - редкие наборы;
  - бесплатные / платные;
  - потраченные рубли;
  - голоса VK.

- Переход в раздел **«Коллекция»** по кнопке со стрелкой.

### Коллекция

- **Последние наборы** — сетка карточек стикеров (`StickerGrid`).
- **Редкие наборы** — список наборов с редкостью выше `common` (rare, epic, legendary), обложка, название, число владельцев.

### Топы

- Рейтинг **TOP-100** коллекционеров (демо-лидерборд).
- Сортировка:
  - по количеству наборов;
  - по потраченным голосам VK;
  - по редким наборам.
- **Поиск** по имени пользователя.
- Топ-3 места выделены отдельным оформлением.

### Статистика

- Карточки: количество наборов, голоса, рубли.
- График **«Рост наборов»** (area chart).
- График **«Голоса и рубли»** (bar chart) по месяцам.

### Профиль

- **Карточка пользователя**: аватар, имя, ссылка на VK, уровень коллекционера.
- Сводная статистика (6 показателей).
- **Карточка коллекционера** (share-card):
  - превью в интерфейсе;
  - кнопка **VK Stories**;
  - кнопка **PNG** — скачивание файла `stickify-card.png`.

## Интеграция с VK

Приложение использует [@vkontakte/vk-bridge](https://www.npmjs.com/package/@vkontakte/vk-bridge):

| Событие VK Bridge | Назначение |
|-------------------|------------|
| `VKWebAppInit` | Инициализация мини-приложения |
| `VKWebAppGetUserInfo` | Имя, фамилия, id и фото пользователя (`photo_200` / `photo_100`) |
| `VKWebAppGetFriends` | Выбор друга для анализа |
| `VKWebAppShowStoryBox` | Публикация карточки в VK Stories |

**Аватар пользователя** загружается из VK и конвертируется в data URL — так фото корректно отображается в интерфейсе и попадает в PNG при экспорте.

> В обычном браузере (`localhost`) VK Bridge недоступен: используются демо-данные и заглушка аватара. Полный функционал (аватар, друзья, Stories) работает при запуске **внутри клиента ВКонтакте**.

## Карточка коллекционера

Вертикальная карточка для соцсетей, формат **1080×1920**.

**Содержимое:**

- логотип и название Stickify;
- аватар и имя коллекционера;
- уровень (например, `Neon Collector` / `VK Collector`);
- сетка из 6 метрик:
  - всего наборов;
  - платные;
  - бесплатные;
  - премиум 100+;
  - потрачено ₽;
  - голоса VK;
- декоративный персонаж на фоне.

  Примеры:
  
<img width="750" height="1450" alt="stickify-story (1)" src="https://github.com/user-attachments/assets/3969825d-0a3b-4604-ad33-58b78a0423c9" />


**Экспорт:** снимается **та же DOM-карточка**, что видна в превью (`html2canvas`), без отдельной вёрстки — PNG совпадает с экраном.

## Технологии

- **React 18** + **TypeScript**
- **Vite 6** — сборка и dev-сервер
- **VKUI 8** — UI-компоненты в стиле ВКонтакте
- **VK Bridge** — API мини-приложения
- **Framer Motion** — анимации переходов и элементов
- **Recharts** — графики на странице статистики
- **html2canvas** — экспорт карточки в PNG

## Запуск локально

### Требования

- **Node.js** 18+ (рекомендуется LTS)
- **npm**

### Установка

```bash
git clone https://github.com/Anyusek/Stickify.git
cd Stickify
npm install
```

### Режим разработки

```bash
npm run dev
```

Приложение откроется по адресу: **http://localhost:5173/** (Может работать с ошибкой или выдавать сообщение "Страница не найдена")

Сервер слушает все интерфейсы (`0.0.0.0`) — удобно для теста с телефона в одной сети.

### Просмотр production-сборки

```bash
npm run build
npm run preview
```

## Демо-данные

Сейчас часть логики работает на **демо-данных** (`src/data/demo.ts`):

- наборы стикеров и статистика после «анализа»;
- графики динамики;
- лидерборд TOP-100.

Реальный запрос к API VK для полного анализа чужих коллекций **ещё не подключён** — `analyzeCollection()` имитирует задержку и подставляет демо-результат с id из ссылки.

При этом **профиль текущего пользователя** (имя, аватар) подтягивается из VK при запуске в клиенте ВКонтакте.
